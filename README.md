#  TP2 - Visualisation des Données

Travail pratique sur la visualisation de données avec Python.

---

##  Description

Ce projet contient deux exercices de visualisation de données :
- **IRIS** : Analyse du célèbre dataset de classification des fleurs
- **TITANIC** : Analyse des passagers du Titanic et facteurs de survie

---

##  Installation

```bash
pip install -r requirements.txt
```

---

##  Contenu du Projet

- `IRIS.ipynb` - Exercice 1 : Dataset IRIS
- `TITANIC.ipynb` - Exercice 2 : Dataset TITANIC
- `requirements.txt` - Dépendances Python

---

##  Exercice 1 : IRIS

**Dataset** : 150 échantillons de 3 espèces d'iris

**Visualisations réalisées** :
- Histogramme des largeurs de pétales
- Distribution de probabilité (KDE)
- Répartition par espèce
- Matrice de corrélation

**Résultat** : Les features permettent une bonne distinction entre les 3 espèces

---

##  Exercice 2 : TITANIC

**Dataset** : 891 passagers du Titanic

**Visualisations réalisées** :
- Répartition hommes/femmes
- Survie par sexe
- Survie par classe
- Relation âge/classe/sexe

**Résultat** : Les femmes et la 1ère classe ont un meilleur taux de survie
